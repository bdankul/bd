MIXING

COLORS

Project Summary

To teach kids how to mix colors.

Audience Profile

Main audience for my website is mainly children but, I would like adults to feel like they could play the game too. Though, it wouldn't be much of a game for them because most adults already know some color mixing.

Perception and Tone

I want the colors to be visually appealing and the site to be as simple as possible. The website should function like a game to make it fun for the user.

Communication Strategy

The pages will be designed simply using only colored shapes and text. Color mixing will be taught using animation and the test in the end, which will play out like a game, will solidify what they've learned.

Target Message

After visiting my site I would hope the kids will be able to mix colors using the colors of the rainbow.
